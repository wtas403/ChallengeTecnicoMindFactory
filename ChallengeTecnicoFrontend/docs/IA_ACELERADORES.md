## Como se uso la IA en este proyecto.

Para agilizar la implementacion se uso el agente de opencode

- Refactor UX/UI del listado de automotores: simplificacion de cabecera, toolbar operativa con refresh, estados visuales, responsive mobile y affordance consistente.
- Implementacion de notificaciones globales tipo toast para altas, ediciones, eliminaciones y errores de API.
- Ajustes de accesibilidad y comportamiento transaccional en formulario y confirmaciones destructivas.
- Soporte de verificacion tecnica sobre tests frontend y build de Angular.
- Correccion UX del buscador para evitar doble control de limpieza en navegadores WebKit y rebalance sutil del layout de la pagina de formulario.
